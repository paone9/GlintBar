GlintBar

A live hardware monitor that sits in the empty gap of your Windows taskbar. No window to keep open, no screen space given up.

Up to twelve metrics, each with a 60-second sparkline, in space the taskbar wasn't using.

What it shows

A single row of tiles, each with a value and a rolling 60-second sparkline. Hover any tile for a larger live graph with the session minimum, maximum and average.

Why it's built this way

It uses space you already lost

The middle of the Windows 11 taskbar is dead space. GlintBar fits itself to that gap, stays out of the way of your taskbar buttons, and yields entirely when the gap gets too narrow or a fullscreen app takes the screen.

No admin rights

Everything comes from sources that don't need elevation: the NVIDIA driver's own nvidia-smi, Windows performance counters, the ACPI thermal zone, and psutil. If you already run HWiNFO or LibreHardwareMonitor, it reads real CPU package temperature and fan RPM from those too — locally, never over the internet.

Source, not a binary

Distribution is source-first with no .exe. Corporate environments block unsigned binaries, and plain Python is easy to audit. Releases are published to PyPI through Trusted Publishing and signed with Sigstore.

It tells you what happened while you were out

Leave the machine and come back to a summary: how long it was actually busy, the typical and peak load, how hot it got, whether memory climbed and stayed up, and which processes used the CPU time.

Pick your tiles, size and alignment. Optional per-second CSV logging for offline analysis.

What you need

  • Windows 10 or 11. It is built on Win32 APIs, so there is no macOS or Linux build.
  • Python 3.10+ and the WebView2 runtime. Windows 11 already has it; Windows 10 usually does via Edge, but may need the free Evergreen runtime.
  • An NVIDIA GPU for the full GPU set. AMD and Intel GPUs report load only; without a GPU those tiles are simply hidden.
  • HWiNFO or LibreHardwareMonitor, optionally, for true CPU package temperature and fan speed.

The only dependencies are pywebview and psutil, installed for you. Configuration and logs live in %LOCALAPPDATA%\GlintBar, never in the install directory.
